Drawing Game

The Drawing Game is an interactive web experience designed to spark creativity by generating unexpected and playful drawing prompts. Instead of simple prompts, the system creates longer randomized instructions that encourage imaginative interpretations and experimental sketches.

Each time the user refreshes or interacts with the interface, a new prompt is generated from a pool of words and phrases. This randomness encourages users to think creatively and approach drawing from new perspectives.

Tools Used: p5.js

Focus: Interaction design, creative prompts, playful user experience
